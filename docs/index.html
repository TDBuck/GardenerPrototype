Gardner Self Publication Platform

Self-publication platform for Treebanks

Download as .zip Download as .tar.gz View on GitHub

The purpose of Gardener was to make something that was simple to use, but powerful enough that it could manage displaying treebanks on its own. Hopefully, anyone, from the large researcher, to the high school class that wants to be proud of a semester's worth of work, can use Gardener, with no knowledge of web design. With this, you should be able to host and display your own collection of treebanks. Tim Buckingham built this as part of the second phase of the Perseids Project.
